# LogicaDE
**LogicaDE** is a trainer-program designed to improve the user's memory, logical thinking and attention.
# Platforms supported
Windows 10+, Linux and MacOS.
# Storyline and gameplay
The player gets dragged into a dungeon by some mysterious magic, the only way to escape from which is to correctly solve various tasks presented on a magical tablet at the end of each location. <br>
**LogicaDE** has 2 gamemodes:
- **Campaign mode** <br>
Consists of 25 logical and mathematical problems, which increase in difficulty as the player progresses further.
- **Freeplay mode** <br>
Has an extended bank of 100 problems which difficulty vary from one to other.
# Installation and launching
1. Download and unpack the .zip archive from the **Releases** tab linked [here](https://github.com/burger7164/LogicaDE/releases).
2. Afterwards, go into the created folder and launch the **LogicaDE.exe** file to launch the game.
